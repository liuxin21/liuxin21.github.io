1. 创建
touch test.c

2. 编辑
nano test.c
或者 （gedit test.c）

```c
# include <stdio.h>
int main()
{
    printf("hello\n");
    return 0;
}
```

3. 编译和链接
gcc -c test.c
gcc -o test test.o
或
gcc -o test test.c

4. 执行
./test


## 执行过程
```
~ xinliu$ cd desktop
desktop xinliu$ mkdir test
desktop xinliu$ cd test
test xinliu$ touch test.c
test xinliu$ nano test.c
test xinliu$ gcc -o test test.c
test xinliu$ ./test
```

1. `test xinliu$ nano test.c` 打开文件
2. `test xinliu$ gcc -o test test.c` 编译文件
3. `test xinliu$ ./test` 执行文件

## 具体内容
`nano test.c`之后编辑内容：

```c
# include <stdio.h>
int main()
{
    int a,b,c;
    printf(" Please enter a value:");
    scanf("%d",&a);
    printf("\n");
    printf("Please enter b value");
    scanf("%d",&b);
    c=a+b;
    printf("\n%d",c);
    return 0;
}
```

## 指针（a = *point_a）
```c
# include <stdio.h>
int main()
{
        int a;
        int *point_a;
        scanf("%d",&a);
        point_a = &a;

        printf("%d\n",a);
        printf("%p\n",point_a);
        printf("%d\n",*point_a);
        return 0;
}
```
